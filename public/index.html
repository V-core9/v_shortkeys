Shortcodes:

1. [Q + W + E] => debugWindowTrigger()

Will trigger visibility of sidebar. Additional: has 3s delay between triggers when continues pressing.

2. [alt + Enter] => triggerFullscreen()

FullScreen trigger function. Additional: has 3s delay between triggers when continues pressing.

3. [alt + C] => closeWindowFunc()

Will just close this tab.
